# Minions frontend

## Tecnologias utilizadas

- ReactJS
- AWS S3
- AWS APIGateway
- AWS Cognito
- AWS SES

* Conexão com os serviços da AWS através do aws amplify (API, Auth e Storage)

* Estilização com styled-components
